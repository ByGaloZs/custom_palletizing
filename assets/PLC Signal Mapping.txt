ROBOT TO PLC
ROBOT STATE: %ID344 INT
0 --> IDLE
1 --> NORMAL OPERATION (PALLETIZING)
2 --> STANDBY 
3 --> MANUAL ACTION (RETIRAR TARIMA)
4 --> MAINTENECE 


VACUUM STATE:
%I38.0 --> VACUUM OK (BIT 0)
%I38.1 --> VACUUM LOST (BIT 1)

PICK MODE: %ID348 INT
0 --> IDLE
1 --> FULL
2 --> HALF-B

PLC TO ROBOT
CONVEYOR STATE:
%Q14.0 --> CONVEYOR READY (BIT 0)

PALLETS STATE:
%Q14.1 --> LEFT PALLET READY
%Q14.2 --> RIGHT PALLET READY
